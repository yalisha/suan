================================================================================
编译方法
================================================================================

方法一:使用编译脚本(推荐)
-------------------------
./compile.sh

方法二:手动编译
---------------
xelatex main.tex
bibtex main
xelatex main.tex
xelatex main.tex

说明:需要运行3次xelatex和1次bibtex以正确处理引用和交叉引用

================================================================================
常见问题
================================================================================

Q: 编译后没有生成PDF?
A: 查看main.log文件中的错误信息: cat main.log | grep Error

Q: 参考文献没有显示?
A: 确保运行了bibtex命令,并且重新编译了2次xelatex

Q: 中文显示乱码?
A: 确认系统已安装华文字体(STSong, STHeiti, STFangsong)

Q: 如何修改内容?
A: 直接编辑对应的.tex文件,然后重新运行compile.sh
