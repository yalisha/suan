## 三、样本选择与数据来源

### (一)数据来源与样本筛选

本研究使用新浪微博热搜榜2024年2月1日至2025年10月1日的历史数据，时间跨度608天，完整覆盖政策实施(2024年11月12日)前后阶段(Pre期253天，Post期355天)。原始数据包含话题名称、热度值、上榜时长、话题分类、时间戳等关键字段。经筛选删除时间戳缺失、空话题、平台未明确分类("其他"类)的记录后，最终样本包含253,227条观测值，对应约215,083个独立话题。

微博作为中国最大社交媒体平台之一(月活超5亿)，热搜榜是用户获取信息的重要入口。根据2023年财报，用户日均使用52分钟，其中热搜浏览15-20分钟，构成注意力分配的核心场景。热搜榜通过算法实时聚合用户行为数据，动态展示50个最受关注话题，是观察算法治理效应的理想样本。

**话题分类**。本研究直接采用微博平台预设的主持人分类字段，包含81个细分类别。选择平台原始分类的理由：(1)减少测量误差，平台分类基于大规模数据训练，一致性高；(2)贴近平台决策逻辑，监管政策实施依赖平台内部分类体系；(3)提升可复制性。聚焦理论假设的核心对比，我们将占比前十的类别(覆盖86%样本)归为两组：**社会类**(包括"社会"标签，占比34.82%)和**娱乐类**(包括"明星"与"明星-内地"，合计16.47%)。社会类涵盖公共事件、民生议题等高舆论风险内容，娱乐类涵盖明星动态、粉丝互动等低风险内容。

### (二)变量定义

**核心因变量**

**H1相关**：(1)**Shannon熵** $H_t = -\sum_{i=1}^{N_t} p_{it} \ln p_{it}$，其中$p_{it}$为话题$i$的热度份额。采用归一化形式$H_t^{\text{norm}} = H_t / \ln N_t \in [0,1]$便于跨日比较。数值越大表示分布越均匀。(2)**HHI指数** $\text{HHI}_t = \sum_{i=1}^{N_t} p_{it}^2$，数值越大表示注意力越集中于少数头部话题。

**H2相关**：(1)**官方媒体占比**，定义为类别$c$在第$t$天上榜话题中官媒主持的比例。官媒包括党媒(人民日报、新华社)、政务账号、传统主流媒体(央视新闻、澎湃新闻)，由两名研究助理独立标注(Cohen's Kappa=0.91)。(2)**信息差指数**，识别标题中的疑问词(为何、为什么)、悬念词(曝、竟、揭秘)、疑问标点，只要出现任一特征即赋值为1，否则为0。(3)**唤醒度指数**，统计强情绪标点(！、？)、极端情绪词(崩溃、泪目、炸了)、强化副词(千万、绝对、最)的总出现次数。采用Jieba分词，关键词预加载为自定义词典以提高准确率。

**H3相关**：**类别注意力份额** $\text{Share}_{ct} = \sum_{i \in c} V_{it} / \sum_{j=1}^{N_t} V_{jt}$，其中$V_{it}$为热度值(微博综合阅读量、讨论量、搜索量的标准化指标)。**类别上榜频次** $\text{Count}_{ct}$为类别$c$在第$t$天的上榜话题数。

**H4相关**：**平均在榜时长** $\text{Duration}_{ct} = \frac{1}{N_{ct}} \sum_{i \in c} d_{it}$，其中$d_{it}$为话题$i$在榜时长(小时)。**注意力密度** $\text{Density}_{ct} = \sum_{i \in c} V_{it} / N_{ct}$，反映单位话题平均热度，密度越高表明类内竞争越不激烈。

**控制变量**：时间趋势$\text{Trend}_t$(1至608)、周内固定效应$\text{DOW}_t$(周一至周日虚拟变量)、月度固定效应$\text{Month}_t$(1-12月虚拟变量)、节假日虚拟变量$\text{Holiday}_t$(法定节假日及其前后一天)、平台总热度$\text{TotalHeat}_t$、上榜话题总数$\text{TotalCount}_t$。

**分组与交互变量**：政策虚拟变量$\text{Post}_t$(若$t \geq$ 2024年11月12日取1，否则0)、社会类虚拟变量$\text{Social}_c$、娱乐类虚拟变量$\text{Entertainment}_c$、官媒虚拟变量$\text{Official}_i$。

### (三)计量模型设定

本研究采用**断点回归时间序列**(ITS)作为主要识别策略。相较于传统DID，ITS更适合全局性政策冲击(所有类别同时受政策影响)、高频时间序列数据(608个日度观测)、需要刻画动态效应的情境。

**模型1：整体去中心化(H1)**
$$Y_t = \beta_0 + \beta_1 \text{Post}_t + \beta_2 \text{Trend}_t + \beta_3 (\text{Post}_t \times \text{Trend}_t) + \boldsymbol{\gamma}' \mathbf{X}_t + \varepsilon_t$$
其中$Y_t \in \{H_t^{\text{norm}}, \text{HHI}_t\}$，$\beta_1$为即时水平变化，$\beta_3$为趋势增量变化。预期$\beta_1 > 0$(熵上升)或$\beta_1 < 0$(HHI下降)。标准误采用Newey-West调整。

**模型2：质量分化(H2主检验)**
$$\text{Official}_t^{\text{Social}} = \alpha_0 + \alpha_1 \text{Post}_t + \alpha_2 \text{Trend}_t + \alpha_3 (\text{Post}_t \times \text{Trend}_t) + \boldsymbol{\delta}' \mathbf{X}_t + u_t$$
预期$\alpha_1 > 0$。**稳健性检验(类内DID)**：在社会类内部以官媒为处理组、非官媒为对照组，
$$Y_{it}^{\text{Social}} = \theta_0 + \theta_1 (\text{Official}_i \times \text{Post}_t) + \theta_2 \text{Post}_t + \theta_3 \text{Trend}_t + \eta_i + \varepsilon_{it}$$
预期$\theta_1 > 0$表明政策后平台更倾向推荐官媒话题。

**模型3：注意力再分配(H3)**
$$\text{Share}_{ct} = \beta_0 + \beta_1^S (\text{Social}_c \times \text{Post}_t) + \beta_2^E (\text{Entertainment}_c \times \text{Post}_t) + \beta_3 \text{Trend}_t + \text{类别FE} + \text{时间FE} + \varepsilon_{ct}$$
预期$\beta_1^S < 0$, $\beta_2^E > 0$, $\beta_2^E - \beta_1^S > 0$(水床效应)。动态扩展用月度虚拟变量替代$\text{Post}_t$以检验超调-回落。

**模型4：密度分化(H4)**
$$Y_{ct} = \gamma_0 + \gamma_1^S (\text{Social}_c \times \text{Post}_t) + \gamma_2^E (\text{Entertainment}_c \times \text{Post}_t) + \gamma_3 \text{Trend}_t + \text{类别FE} + \text{时间FE} + u_{ct}$$
其中$Y_{ct} \in \{\text{Duration}_{ct}, \text{Density}_{ct}\}$，预期$\gamma_1^S > 0$, $\gamma_2^E < 0$。

**识别假设**：在无政策干预的反事实下，结果变量沿既有趋势演化。威胁包括同期事件、季节性混淆、自相关，通过事件核查、固定效应控制、稳健标准误应对。稳健性检验包括安慰剂检验(虚假断点)、替代指标(Gini系数、Top10占比)、样本敏感性、模型设定稳健性、平行趋势检验。
